TODO: Benutzerhandbuch zur Verwendung Ihrer Anwendung Ablauf: Das Spiel "High Speed Racer 2D" ist ein 2D-Geschicklichkeitsspiel mit dem Ziel, möglichst lange den entgegen kommenden Fahrzeugen auszuweichen. Dabei ist der Score abhängig von der zurückgelegten Strecke. Der Spieler hat nur ein Leben. Stößt der Spieler mit einem Objekt zusammen, so stoppt das Spiel und der Score. Steuerung: Das Auto wird vom Spieler mithilfe der Pfeiltasten Rechts-nach rechts, Links-nach links, Oben-nach oben, Unten-nach unten gesteuert.
